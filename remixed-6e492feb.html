The Problem

Load assessment is the most
manual step in the workflow.

1–3 days

Per project, every project

Dead loads calculated layer by layer. Live loads looked up by hand from code tables. Wind and seismic worked through code formulas. Then re-entered manually into FEM software. This happens on every single project, regardless of how similar it is to the last one.

0 checks

On the data transfer

Numbers copied from Excel into SAP2000 one by one. Wrong units, transposed digits, missed load cases — errors that enter silently and propagate through the entire design. There is no validation step between the load schedule and the FEM model. There should be.

∞ times

Repeated from scratch

Every time the architect revises the floor plan, the structural system changes, or a new scope is added — the load schedule has to be rebuilt manually. The same engineer, doing the same work, inside another inherited spreadsheet.

0%

Engineering judgment required

Load assessment follows rules — design code rules, material density tables, occupancy classifications. It is not creative work. It does not require a licensed engineer's intuition. It requires precision and consistency, which is exactly what software does better than people.

The Product

What Civil Agent does

Civil Agent is an AI-powered load takeoff tool built specifically for structural engineers. You give it your building file and a few project parameters. It gives you a fully verified load schedule, a professional calculation report, and a file you can import directly into SAP2000 or ETABS — without touching a single spreadsheet.

📂
Input — what you provide
🏛

Building file

Upload an IFC file exported from Revit or ArchiCAD, or a DWG from AutoCAD. Civil Agent reads the floor geometry, room names, occupancy zones, and building dimensions automatically.

📍

Project location

Tell us where the building is. Civil Agent uses location to pull design wind speed from code wind maps and seismic zone data — no manual lookup.

📘

Design code selection

Select your applicable code — Eurocode, ASCE 7, or BS 6399. Civil Agent applies the correct load values, partial factors, and load combination rules for that code throughout.

🧱

Floor buildup

Choose from standard floor buildup configurations — structural slab, screed, finish, raised floor, ceiling, services. Or define a custom buildup for specialist zones like plant rooms or terraces.

Output — what you receive
📊

Complete load schedule

A full table of dead loads, live loads, and imposed loads for every floor zone in the building — calculated automatically from your building file and buildup selections. Every value traceable to its source.

💨

Wind & seismic forces

Wind pressures per facade and equivalent floor-level lateral forces. Seismic base shear distributed across floor levels. All calculated per your design code from the building geometry and location.

⚙️

All load combinations

Every ULS and SLS load combination required by your design code — automatically assembled and ready to apply. Each combination references the exact code clause that defines it.

📤

FEM import file + PDF report

A structured import file for SAP2000 or ETABS — loads applied to the right elements automatically. Plus a professional PDF calculation report for submission, checking, and your project record.
